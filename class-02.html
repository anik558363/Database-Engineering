What is a Database Mangagement System (DBMS)? - A Database Management System (DBMS) is software - Interacts with users, applications - Providing tools for data organization, storage, and Administration 2. Key Features of a DBMS - Data Storage - Data Retrieval - Data Manipulation - Data Security - Data Integrity - Backup and Recovery 3. Relational Database Management System (RDBMS) - Organizes data into tables (rows and columns) with relationships between tables. - Use SQL (Structured Query Language) for interacting with the database. - Examples: MySQL, PostgreSQL, Oracle, Microsoft SQL Server - Structured Data 4. NoSQL Database Management System - Unstructured or semi-structured data - NoSQL systems can store data as key-value pairs, documents, graphs, or wide-column stores. - Examples: MongoDB, Cassandra, Redis, Couchbase 5. Advantages of Using a DBMS - Data Independence - Efficient Data Management - Data Security - Data Integrity - Backup and Recovery 6. Popular DBMS Systems - MySQL: Open-source RDBMS, widely used for web applications. - PostgreSQL: Open-source RDBMS, known for its advanced features and extensibility. - Oracle Database: Commercial RDBMS, widely used in enterprise applications. - Microsoft SQL Server: Commercial RDBMS, commonly used in Windows environments. - MongoDB: Open-source NoSQL database, designed for scalability and flexibility.
